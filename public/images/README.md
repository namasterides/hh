# Heritage Honey — Image Drop Folder

Drop your real food photos here using these exact filenames. Until a file
exists, the site renders a graceful cream/terracotta placeholder with the
H monogram, so the layout never breaks.

| Filename     | Used in            | Recommended size            |
| ------------ | ------------------ | --------------------------- |
| `hero.jpg`   | Hero background    | 2000×1200+ (landscape)      |
| `about.jpg`  | About section      | 1000×1250 (portrait 4:5)    |
| `dish-1.jpg` | Gallery tile 1     | ~1200×1200 (square/portrait)|
| `dish-2.jpg` | Gallery tile 2     | ~1200×1200                  |
| `dish-3.jpg` | Gallery tile 3     | ~1200×1600 (portrait)       |
| `dish-4.jpg` | Gallery tile 4     | ~1200×1200                  |
| `dish-5.jpg` | Gallery tile 5     | ~1200×1200                  |
| `dish-6.jpg` | Gallery tile 6     | ~1200×1600 (portrait)       |
| `dish-7.jpg` | Gallery tile 7     | ~1200×1200                  |
| `dish-8.jpg` | Gallery tile 8     | ~1200×1200                  |

JPG or WebP both work — just keep the `.jpg` filename (or update the paths
in `src/components/Hero.tsx`, `About.tsx`, and `Gallery.tsx`).
